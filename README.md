## O que é?

**AIara** é uma assistente de IA **100% offline** para gerar scripts de programação. Interface estilo ChatGPT, mas funciona sem internet!

## Scripts Disponíveis

| Linguagem | Scripts |
|-----------|---------|
| **Lua (Roblox)** | Sistema de Voo, Spawner de Armas, Template Genérico |
| **Python** | Calculadora de Média, Template Genérico |
| **Java** | Sistema Bancário (POO), Template Genérico |

## Como Usar

### Online (GitHub Pages)
1. Acesse: `https://SEU-USUARIO.github.io/AIara/`
2. Digite o que você quer criar
3. Clique no botão verde ou aperte **Enter**
4. Copie o código com o botão "Copiar"

### Offline (Local)
1. Baixe esta pasta
2. Abra `index.html` no navegador
3. Funciona 100% sem internet!

## Estrutura

```
AIara/

├── css/
```

## Palavras-Chave

Mencione estas palavras para obter scripts específicos:

- **"voo"**, **"voar"**, **"fly"** → Script de voo para Roblox
- **"arma"**, **"spawner"**, **"weapon"** → Spawner de armas Roblox
- **"média"**, **"nota"**, **"escola"** → Calculadora Python
- **"banco"**, **"bancário"**, **"conta"** → Sistema bancário Java
- **"roblox"**, **"lua"** → Template Lua genérico
- **"python"** → Template Python genérico
